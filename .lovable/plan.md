

# Akhilesh Somani — Author Website for *The 9 Wonders*

## Design System
- **Colors:** Off-white background (#F9F9F9), warm greys, muted gold accent for CTAs, deep charcoal for text
- **Typography:** Playfair Display (serif) for headings, Inter (sans-serif) for body text — loaded via Google Fonts
- **Style:** Luxury minimalist with generous whitespace, soft transitions, and fade-in-on-scroll animations

---

## Page Sections (Single-Page Layout)

### 1. Sticky Navigation Bar
- Translucent/frosted-glass effect, stays fixed at top
- Logo/name "Akhilesh Somani" on left, smooth-scroll links on right (The Book, About, Testimonials, Contact)
- Mobile hamburger menu

### 2. Hero Section
- Centered or split-screen layout
- 3D-styled book cover mockup of *The 9 Wonders* (placeholder image)
- Compelling tagline and subtitle
- "Buy Now" gold accent CTA button with soft opacity hover transition
- Fade-in entrance animation

### 3. The Concept — "The 9 Wonders"
- Clean 3-column grid (stacks on mobile) showcasing 9 thematic cards
- Each card: icon/number, wonder title, one-line description
- Subtle fade-in-on-scroll for each card

### 4. Author Bio — "About Akhilesh"
- Two-column layout: professional headshot placeholder + elegant biography text
- Minimalist styling with warm grey accents
- Fade-in animation on scroll

### 5. Social Proof
- Testimonial slider with reader/reviewer quotes
- Optional "Featured In" grayscale logo bar
- Smooth auto-advancing carousel

### 6. Newsletter — "Join the Journey"
- Single email input field with gold "Subscribe" button
- Clean centered layout with short motivational copy
- Frontend-only (no backend wired up initially)

### 7. Footer
- Social media icon links (Instagram, Twitter/X, LinkedIn, etc.)
- Contact email, copyright notice
- Clean, minimal design

---

## Interactions & Polish
- Scroll-triggered fade-in animations on all sections
- Soft opacity hover transitions on all buttons and links
- Smooth scroll navigation between sections
- Fully responsive across desktop, tablet, and mobile
