# Akhilesh Somani — The 9 Wonders

Discover 'The 9 Wonders' by Akhilesh Somani — a transformative journey through nine timeless principles that shape extraordinary lives.

## Technologies Used

- **Vite**
- **TypeScript**
- **React**
- **shadcn/ui**
- **Tailwind CSS**
- **Framer Motion**

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Project Structure

- `src/components`: Reusable UI components.
- `src/lib`: Utility functions and configurations.
- `src/pages`: Main application pages.
- `public`: Static assets like favicons and images.
